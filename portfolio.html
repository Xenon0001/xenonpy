GES - Gestor Escolar

GES (Gestor Escolar) es un sistema integral desarrollado para digitalizar y optimizar los procesos administrativos y académicos en instituciones educativas de Guinea Ecuatorial. La plataforma permite gestionar matrículas, calificaciones, asistencia, horarios y comunicación entre docentes, estudiantes y padres de familia.

Desarrollado con tecnologías modernas y adaptado a la realidad de las escuelas en el país, GES ha demostrado ser una herramienta invaluable para mejorar la eficiencia y transparencia en la gestión educativa.

Tecnologías utilizadas

Python Django PostgreSQL HTML5/CSS3 JavaScript

Educación

Aprende Python desde Guinea

"Aprende Python desde Guinea" es una plataforma de formación en programación diseñada específicamente para jóvenes ecuatoguineanos. El curso parte desde cero y utiliza ejemplos y proyectos relevantes al contexto local, haciendo el aprendizaje más accesible y significativo.

La plataforma incluye lecciones interactivas, ejercicios prácticos, proyectos reales y una comunidad de apoyo. Hasta la fecha, más de 50 estudiantes han completado el programa, muchos de los cuales han conseguido empleo en el sector tecnológico o iniciado sus propios emprendimientos.

Tecnologías utilizadas

Python JavaScript HTML5/CSS3 Flask SQLite

Salud

Salud Conectada

Salud Conectada es un sistema diseñado para mejorar la comunicación y gestión de pacientes entre hospitales y centros de salud en Guinea Ecuatorial. La plataforma permite el registro centralizado de pacientes, historiales médicos digitales, programación de citas y seguimiento de tratamientos.

Desarrollado en colaboración con profesionales de la salud locales, el sistema respeta las normativas de privacidad y está adaptado a la infraestructura tecnológica disponible en el país. Actualmente en fase piloto en dos hospitales de Malabo, con planes de expansión a nivel nacional.

Tecnologías utilizadas

Python Flask MySQL JavaScript Bootstrap
